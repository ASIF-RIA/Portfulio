## Option 2: Clone to a New Location

If you want to clone the repository to a different folder:

```bash

cd C:\Users\Asif\OneDrive\Desktop

# Clone the repository
git clone https://github.com/ASIF-RIA/Portfulio.git

# This will create a new folder called "Portfulio"
# Navigate into it

```
