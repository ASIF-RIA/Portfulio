## ðŸ“ Customization Guide

### Update Profile Information

1. **Name and Title**: Edit the hero section in `index.html` (around line 850)
   ```html
   <h1>Your Name</h1>
   <p>Your Title</p>
   ```

2. **Profile Image**: Replace `profile.jpg.jpeg` with your own image. Make sure to keep the same filename or update the reference in `index.html`.

3. **About Section**: Update the about me content (around line 800)

4. **Contact Information**: Update contact links in the contact section (around line 1258)
   - Email
   - GitHub
   - LinkedIn
   - Facebook

### Update Projects

Edit the projects section in `index.html` (around line 975):
- Add or remove project cards
- Update project descriptions, technologies, and details

### Update Skills

Modify the skills section (around line 1110):
- Add or remove skill cards
- Update skill descriptions

### Change Colors

The color scheme uses CSS variables defined at the top of the `<style>` section (around line 10):
```css
:root {
    --primary: #2563eb;    /* Primary blue color */
    --secondary: #10b981;  /* Secondary green color */
    --accent: #f59e0b;     /* Accent orange color */
    /* ... other colors ... */
}
```

### Update Education

Modify the education timeline section (around line 856) to reflect your academic journey.
